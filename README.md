# lamsey_env

quick env configs for new machines. very much wip.

# Usage

## Ubuntu 22

The following installs basic tools like python, GPU drivers, etc. with confirmation prompts for each group of software.

```bash
cd lamsey_env
./setup_ubuntu22.sh
```

## OSX or other Unix

Ubunutu colors for your terminal:

```bash
cd lamsey_env
./config_bashrc.sh
```
